☁️ Sincronizar entre dispositivos (opcional)

Mantén tu inventario igual en el celular y el PC. Usa Supabase (gratis). Ver cómo configurarlo ▾

  1. Crea una cuenta gratis en supabase.com y un proyecto nuevo.
  2. En el proyecto: SQL Editor → pega y ejecuta esto: 
    create table if not exists inventario_myl (
  clave text primary key,
  datos jsonb,
  actualizado timestamptz default now()
);
alter table inventario_myl enable row level security;
create policy "acceso" on inventario_myl
  for all using (true) with check (true);
  3. En Settings → API copia Project URL y la clave anon public.
  4. Pégalas abajo y elige una clave de colección (un código tuyo, p. ej. tu correo o una palabra secreta). Úsala igual en todos tus dispositivos.

Opcional — para el historial de guardados (quién y cuándo), ejecuta también este SQL una vez:

create table if not exists inventario_myl_log (
  id bigserial primary key,
  clave text,
  dispositivo text,
  accion text,
  copias int,
  creado timestamptz default now()
);
alter table inventario_myl_log enable row level security;
create policy "acceso_log" on inventario_myl_log
  for all using (true) with check (true);

Opcional — para tiempo real (que se actualice solo entre pantallas abiertas), ejecuta esto una vez. Es gratis en el plan Free de Supabase:

alter publication supabase_realtime add table inventario_myl;
